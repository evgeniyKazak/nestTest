## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Migrations

```bash
# generate first migraion
npm run migration:generate db/migrations/build

# run
npm run migration:run
```
